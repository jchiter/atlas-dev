== Description ==
Homelist is a Real Estate WordPress theme designed for real estate agencies, agents and property owners. You can also use this theme as a corporate page. The homepage has many sections: full width slider, comprehensive property search filters, locations, testimonials and other necessary options to present information about the services you provide. 
With Homelist – Real Estate WordPress Theme users can manage their own properties without accessing admin dashboard. They can manage their profile, password and property listing. 
You can add as much agent as you need for a property. 
By using Homelist you can start charging your users and earn money. We prepared for you theme with strong monetization focus. Pay per post and packages are industry standard but you can use our new "extras" earning tools (including pay per featured or pay for TOP) to increase your monthly wage. Combination of monetization and our statistics makes ApusHome a powerful theme.
It has a fully responsive width adjusts automatically to any screen size or resolution.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Homelist in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
